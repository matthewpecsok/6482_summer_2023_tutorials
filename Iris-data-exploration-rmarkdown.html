1 Data description

When learning a new technique its often helpful to start with a dataset that is well known and that has characteristics useful in the context of the technique. The IRIS dataset is one of the best datasets for introductory model building because it has 3 classes, one of which is linear separable and the other two are not.

https://en.wikipedia.org/wiki/Iris_flower_data_set

The target variable is the class of the flowers. A column called species

We will use this notebook to learn the attributes of the dataset.

In addition pay CLOSE attention to the code syntax throughout. You will need to learn this syntax quickly.

4.2.1 Numbered bullets

here we show data split into bins by

  1. 25% increments/bins (Quartiles)
  2. 20% increments/bins (Quintiles or Pentiles)
  3. 10% increments/bins (Deciles)

For the curious student there are many different quantiles. Learn more names here. https://en.wikipedia.org/wiki/Quantile#:~:text=Common%20quantiles%20have%20special%20names,than%20for%20the%20cut%20points.

iris$Sepal.Length %>% quantile()

## proportion plots

This is just one of many ways to calculate the proportions in R and tidyverse.

iris %>%
  count(Species) %>%
  mutate(total_n=sum(n)) %>%
  mutate(prop = round(n/total_n,2)) %>%
  ggplot() +
  geom_col(aes(x=Species,y=prop,fill=Species)) +
  ggtitle('barplot of Species')
